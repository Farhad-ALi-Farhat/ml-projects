# 🤖 Machine Learning Projects (Python)

This repository contains machine learning and deep learning projects built using Python, NumPy, Scikit-learn, and TensorFlow.

The goal of this repo is to demonstrate both:
- Practical ML implementations
- Core concepts built from scratch

---

## 📂 Projects

### 📧 Spam Classifier
- NLP-based classification
- Uses CountVectorizer + Naive Bayes
- Predicts whether a message is spam or not

---

### 📈 Linear Regression (From Scratch)
- Implemented using NumPy only
- Uses Gradient Descent
- No external ML libraries

---

### 📉 Gradient Descent (From Scratch)
- Core optimization algorithm used in ML
- Minimizes error using iterative updates
- Demonstrates:
  - Cost function (MSE)
  - Gradient calculation
  - Parameter updates

---

### 🔢 Digit Classifier (Neural Network)
- Uses MNIST dataset
- Built with TensorFlow/Keras
- Fully connected neural network

---

### 📊 K-Means Clustering
- Unsupervised learning algorithm
- Groups data into clusters
- Visualized using Matplotlib

---

## 🧠 Concepts Covered

- Supervised Learning
- Unsupervised Learning
- Neural Networks
- Gradient Descent Optimization
- Loss Functions (MSE)
- Feature Extraction (NLP)
- Model Evaluation

## 🚀 How to Run

Clone the repository:

```bash
git clone https://github.com/your-username/ml-projects.git
cd ml-projects
```

Run any project:

```bash
python filename.py
```
